# Manager Software Engineering Master Curriculum Specification

## Goal

Build a professional interactive offline HTML learning portal for a 12+
week preparation journey targeting Manager Software Engineering roles at
Mastercard.

## Requirements

-   12+ week curriculum
-   2--3 hours/day
-   Blend technical, engineering management, and payments every day.
-   Beginner-friendly where needed.
What the final deliverable would be
A single professional HTML application (works offline) containing 12+ weeks of study material, equivalent to a 250–400 page handbook.
